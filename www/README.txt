This folder contains the necessary files to host a small web application to 
showcase the master project and to experiment with the trained models.

Files / Directories:
    - app.py: flask web application
    - start_webapp_development.sh: start the web app for development

    - static: files the webapp serves
    - models: host pre-trained classifiers

How to run:
    - To launch the webapp in development:
        - bash start_webapp_development.sh
